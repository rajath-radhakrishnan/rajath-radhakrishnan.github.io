Tools and Resources

The following is a collection of tools and websites that I find useful for my academic research.

Tools

  • arXiv Digest: A curated list of arXiv papers on TQFT and related topics by Clement Delcamp.

  • Detexify: Draw characters to get LaTeX code.

  • GAP: Software for computational group theory.

  • Jupyter: Notebook for programming languages. (including SAGE and GAP)

  • Kac: Software for computing RCFT data.

  • Mathcha: Useful for drawing figures and exporting it to TikZ and SVG.

  • Mendeley: Online reference manager.

  • QuillBot: English writing help.

  • Quiver: Quiver and commutative diagram drawing.

  • SageMath: Software for computational mathematics.

  • SciRate: Website for rating and commenting on arXiv preprints.

  • WhiteboardFox: Simple whitebaord. Send link to collaborators to collaborate.

  • When2meet: Survey to fix date and time for a large group meeting.

Resources

  • Anyon Wiki: Wiki on anyon models.

  • ATLAS of Finite G: Finite group representation theory data.

  • Atlas of Fusion Rings: Fusion rings of small rank and SageMath codes.

  • Cambridge Part III course notes by Dexter Chua.

  • Error Correction Zoo: Catalog of classical and quantum error correcting codes.

  • Expository papers by Keith Konrad.

  • IBM Quantum Learning: Online resource for learning quantum computation using Qiskit.

  • Kerodon: Online textbook on homotopy theory.

  • KnotInfo: Table of knots.

  • Manifold Atlas Project: Wiki for manifolds.

  • Math notes by J.S Milne.

  • Math Wiki.

  • nLab: Wiki for category theory.

  • PennyLane: Online resource for learning quantum computation using PennyLane.

  • The Stacks Project: Online textbook and reference on algebraic geometry.

  • Topological Quantum: Online textbook by Steve Simon.

  • VOA and MTC: Online database of VOA and MTC data.


Note: Please let me know if a link is broken.
